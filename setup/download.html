Setup: Download/Install/Register projects listed below

Example code, fork these on GitHub:

  • RhymesSite repo (for site developer exercises)
  • Rhymes Distro repo (for project maintainers and release masters)
  • Rhymes module repo (for project maintainers and release masters)

For basic local development you need:

  • *AMP for local Drupal install
  • Drush
  • Git

Site developers collaborating on GitHub and Acquia Cloud need:

  • GitHub user account
  • Free Acquia Cloud developer site
  • Drush aliases (optional/recommended)

Project maintainers managing contrib projects inside a site repo need:

  • Git subtree (Tip: Install on Mac via Homebrew)
  • Drush Subtree (optional, only works with Drush master/7.x branch)

Release masters preparing production builds from versioned external projects need:

  • Drush versions:
    • 5.x patched here (patch pending #836)
    • 6.x patched here (patch pending #834)
    • 7.x/master
  • Some familiarity with Drush Make
  • Build Manager (optional/experimental, requires Drush 7.x)
